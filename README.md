# The Farm That Gives Back

A quick hackathon prototype for Apron Creative.

The app is a one-screen interactive awareness game: visitors restore a bare patch of land into a regenerative community farm. Each action raises biodiversity, climate resilience, and community wellbeing, then points people to Apron's Crowdfunder.

## Run locally

```bash
npm start
```

Open `http://localhost:3000`.

## Railway

This project is intentionally static and dependency-free. Railway can run it with:

```bash
npm start
```
